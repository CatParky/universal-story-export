📋 How to Export Your Claude Conversations

Method 1: GitHub Browser Script (Recommended)

Export individual conversations as Markdown files:

  1. Go to github.com/CatParky/claude-chat-exporter
  2. Click on claude-chat-exporter.js
  3. Copy all the code (click "Copy raw file" button)
  4. Open your Claude conversation in claude.ai
  5. Press F12 (or Cmd+Option+J on Mac) to open Developer Console
  6. Paste the script in the Console tab and press Enter
  7. Your conversation will download as a .md file!

💡 This is a personal fork for reliability. Original script: agarwalvishal/claude-chat-exporter

Method 2: Claude Official Export

Export all your conversations at once:

  1. Go to Settings → Privacy in Claude.ai
  2. Click "Export data"
  3. Wait for email with download link
  4. Download the ZIP file
  5. Extract and upload individual JSON files here
