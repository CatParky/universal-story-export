Convert saved chat exports for use with Story Extractor

Version 2.1 • 1 April 2026

📋 How to Use

  1. Save your chat page as MHTML (File → Save Page As → Webpage, Single File)
  2. Required: Enter the character names EXACTLY as they appear in the chat
  3. Drop the .mhtml file below or click "Choose File"
  4. The converted JSON will download automatically
  5. Upload the JSON to Story Extractor

💡 How the Name Fields Work:
• Character Name - copy EXACTLY from your chat (used for matching)
• Show Name As - clean version for the JSON output (optional)

Examples:
• Match: "💜 Gale Dekarios [Only One Bed]" → Show As: "Gale Dekarios"
• Match: "River 'Riv'" → Show As: "Riv High-Hill"
• Match: "Theo" → Show As: "Theo Stirling"
